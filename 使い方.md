# ToDoリストを「常に最前面」で使う手順

`todo.hta`（インストール不要の独立アプリ）を開き、PowerToysで最前面に固定します。
付箋（スティッキーノート）のように、他のアプリの手前へ常に表示できます。

## 用意するもの
- `todo.hta` … ToDoリスト本体（これ1つだけでOK）
- PowerToys … 最前面固定に使用（インストール済み）

> タスクは `todo.hta` と同じフォルダの `todo-data.json` に自動保存されます。
> 別PCへ移すときは、この2ファイルを一緒にコピーすればデータも引き継げます。

---

## 手順1：ToDoリストを開く
1. `todo.hta` を好きなフォルダ（デスクトップなど）に置く
2. **ダブルクリック** して起動
   → タブもアドレスバーもない、小さな専用ウィンドウが開きます

> ※ 初回に「実行しますか？」と出たら「実行」を選んでください
> （自分で作ったファイルなので問題ありません）。

---

## 手順2：最前面に固定する（PowerToys）
1. PowerToys が起動していることを確認
   （設定 →「常に最前面 (Always on Top)」がオンになっていること）
2. ToDoリストのウィンドウをクリックして選択
3. **Win + Ctrl + T** を押す
   → ウィンドウの枠が光り、常に手前に表示されるようになります
   （もう一度 Win + Ctrl + T で解除）

あとは好きな位置・大きさに動かして、画面のすみに置けば付箋のように使えます。

---

## （任意）PC起動時に自動で開く
1. `Win + R` を押して `shell:startup` と入力 → Enter
2. `todo.hta` の **ショートカット** を作り、開いたフォルダに入れる
   （todo.hta を右クリック →「ショートカットの作成」→ 移動）

これでPC起動時に自動でToDoリストが開きます。
※ 最前面固定（Win + Ctrl + T）は起動後に一度押してください。

---

## メモ・トラブル対処
- 入力したタスクは `todo-data.json` に保存されます。誤って消すとタスクも消えます。
- todo.hta が開かないときは、右クリック →「プログラムから開く」→
  `Microsoft (R) HTML アプリケーション ホスト（mshta）` を選んでください。
- ブラウザで使いたい場合は、同梱の `todo.html` も同じ機能で使えます
  （その場合も、ブラウザのウィンドウを Win + Ctrl + T で最前面固定できます）。
