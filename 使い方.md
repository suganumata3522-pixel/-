# ToDoリストを「常に最前面」に表示する手順

付箋（スティッキーノート）のように、他のアプリの手前に常にToDoリストを表示する設定です。

## 用意するファイル
- `todo.html` … ToDoリスト本体
- `todo-start.bat` … タブのない独立ウィンドウで開くための起動ファイル

この2つを **同じフォルダ**（例：デスクトップ）に置いてください。

---

## 手順1：PowerToys をインストール（最前面に固定するため）
Windowsには「常に最前面」機能が標準でないため、Microsoft公式の無料ツール **PowerToys** を使います。

1. Microsoft Store を開く →「PowerToys」で検索 → インストール
   （または https://learn.microsoft.com/windows/powertoys/install からダウンロード）
2. PowerToys を起動し、左メニューの **「常に最前面 (Always on Top)」** がオンになっていることを確認

> ショートカットキー：**Win + Ctrl + T**（対象ウィンドウを最前面に固定／解除）

---

## 手順2：ToDoリストを開く
- `todo-start.bat` を **ダブルクリック** します。
- アドレスバーやタブのない、小さなウィンドウでToDoリストが開きます（付箋のような見た目）。

> ※ 初回に「WindowsによってPCが保護されました」と出たら、
> 「詳細情報」→「実行」で開けます（自分で作ったファイルなので安全です）。

---

## 手順3：最前面に固定する
1. ToDoリストのウィンドウをクリックして選択
2. **Win + Ctrl + T** を押す
3. ウィンドウの枠が青く光り、常に手前に表示されるようになります
   （もう一度 Win + Ctrl + T で解除）

好きな位置・サイズに動かして、画面のすみに置いておけば付箋のように使えます。

---

## （任意）PC起動時に自動で開く
毎回手動で開くのが面倒な場合：

1. `Win + R` を押して `shell:startup` と入力 → Enter
   （スタートアップフォルダが開きます）
2. `todo-start.bat` の **ショートカット** を作成して、このフォルダに入れる
   （bat を右クリック →「ショートカットの作成」→ できたショートカットを移動）

これでPCを起動するたびに自動でToDoリストが開きます。
※ 最前面固定（Win + Ctrl + T）は起動後に一度押す必要があります。

---

## メモ
- 入力したタスクは、そのPCのブラウザに自動保存されます。
- `todo-start.bat` が動かない場合は、`todo.html` を直接ダブルクリックしても使えます
  （その場合は通常のブラウザのタブで開きます）。
