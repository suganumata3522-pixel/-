断面性能計算ツール（長方形・円形・H形鋼）

断面の寸法を入力するだけで、断面積 A・断面二次モーメント I・断面係数 Z・断面二次半径 i を自動計算します。H形鋼は強軸（x）・弱軸（y）の両方を計算します。課題の検算や実務での概算チェックにご利用ください。

計算式

項目長方形（b × h）円形（直径 D）
断面積 Ab × hπD² / 4
断面二次モーメント Ibh³ / 12πD⁴ / 64
断面係数 Zbh² / 6πD³ / 32
断面二次半径 i√(I / A)

H形鋼の計算式（せい H・幅 B・ウェブ厚 t1・フランジ厚 t2）

項目強軸（x 軸）弱軸（y 軸）
断面積 A2·B·t2 ＋ (H − 2·t2)·t1
断面二次モーメントIx ＝ [ B·H³ − (B − t1)·(H − 2·t2)³ ] / 12Iy ＝ [ 2·t2·B³ ＋ (H − 2·t2)·t1³ ] / 12
断面係数Zx ＝ Ix / (H/2)Zy ＝ Iy / (B/2)
断面二次半径ix ＝ √(Ix / A)iy ＝ √(Iy / A)

H形鋼はフィレット（フランジ付け根の丸み）を無視した概算値です。正確な値は鋼材の形鋼表（断面性能一覧）を確認してください。公式の意味は「H形鋼の断面二次モーメント（強軸・弱軸）」「断面二次モーメントとは？」で解説しています。

ご注意

本ツールの計算結果は学習・検算の参考用です。実際の設計には使用せず、構造計算は法令・基準にもとづき責任ある技術者の判断のもとで行ってください。
