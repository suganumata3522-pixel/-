# 球場 (本拠地) 背景画像配置ディレクトリ

各球団のスライド・サムネ背景に使う本拠地球場の写真をここに配置します。
ファイルが存在する球団については、Pexels / AI 生成より優先してこの画像が背景になります。
ファイルが無ければ従来通り `images.sources` (local → pexels → ai_gen) から取得します。

## 推奨フォーマット

- 横長 16:9 (1920x1080 以上)
- スタンドからの俯瞰や、グラウンド全景など「球場と分かるカット」
- 文字・ロゴ・観客アップが画面の中央~下部に大量に乗っているものは避ける
  (タイトル帯・本文パネルが上に乗るため)

## 著作権について (重要)

スタジアム写真も撮影者の著作物です。自分で撮影したもの、または
権利者から動画利用許諾を得た素材だけを置いてください。
ファイルが無い球団については、Pexels (商用利用可) または AI 生成画像が
代替として使われます。
