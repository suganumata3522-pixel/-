- PNG (透過推奨)
- 短辺 600px 以上
- 余白を切り詰めた状態のロゴ単体

## 著作権について (重要)

球団ロゴ・標章は各球団・親会社の登録商標および著作物です。
このパイプラインで合成した動画を **公開する場合は、各球団の利用規約・素材利用ポリシーを必ず事前に確認してください。**
規約上の許諾範囲外の使用 (営利目的のYouTube収益化など) は権利侵害になり得ます。

ファイルが存在しない球団については、サムネ/スライド中央に「球団名テキスト」が代わりに表示されます (色は球団テーマカラー)。
