# Pikmin Throw (ファンメイド習作)

React Native (Expo) で作られたピクミン風モバイルシューティングゲームです。

> ⚠️ **権利について**：ピクミン / オリマーは任天堂株式会社の著作物です。本プロジェクトは個人の学習目的のファンメイド習作であり、配布・販売・ストア公開は行わないでください。

## ゲーム内容

- プレイヤーは画面下部の **オリマー風キャラ**
- 画面下のボタンで投げるピクミンの色を選択
- 画面を指でなぞった方向にピクミンを投げる
- ピクミンは敵に**張り付いて**ダメージを与える（離れない）
- 群がるほど早く倒せる（張り付き数 = 1秒あたりのダメージ × 2）

## 要素システム（属性相性）

| 敵 | 色 | 攻撃可能なピクミン |
|----|----|---|
| 通常 | 赤 | **全色OK** |
| 火（オレンジ＋炎） | オレンジ | **赤のみ** |
| 電気（黄色＋稲妻） | 黄 | **黄のみ** |
| 水（青＋水滴） | 青 | **青のみ** |
| **ボス**（大型・暗赤） | 暗赤 | **全色OK / HP 25** |

違う色を当てるとピクミンが弾かれて消滅します（要振動）。

## スコア

- 通常敵撃破：**30点 + 張り付き数 × 10点**
- ボス撃破：**100点 + 張り付き数 × 10点**

## 操作

| 操作 | 効果 |
|------|------|
| 画面（上部）をタップ・ドラッグ | オリマーがX座標に追従＋指の位置に向けて投げる |
| 下部の色ボタン | 投げるピクミンの色を切り替え |
| 何もしない | まっすぐ上に投げる |

## 振動フィードバック (iOS/Android のみ)

- 張り付き成功：30ms
- 違う色で弾かれる：15ms
- 敵撃破：80ms
- 自機被弾：120ms

## セットアップ

```bash
npm install
npm start
```

Expo Go アプリ（iOS / Android）で QR コードを読み取るとプレイできます。
ブラウザで試す場合は `npm run web` を実行してください（振動は動作しません）。

## 技術スタック

- React Native 0.73
- Expo SDK 50
- TypeScript
- すべてのキャラは View シェイプで描画（外部画像なし）
