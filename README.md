# 電脳せどり管理ツール

利益商品の自動リサーチから販路設定、購入日の決定判断、仕入・売上管理までを
一元管理できるWebアプリです(Flask + SQLite、ローカル動作)。

## 機能

| 機能 | 内容 |
| --- | --- |
| 📊 ダッシュボード | 今月の売上・純利益・仕入額・在庫・候補数、月次推移、次のおすすめ購入日 |
| 🔍 商品リサーチ | 楽天市場 / Yahoo!ショッピングAPIを横断検索し、販路手数料・ポイント還元込みで利益・利益率・ROIを自動試算 |
| 🤖 自動リサーチ | 保存キーワードを一括検索し、利益条件(最低利益額・利益率・ROI)を満たす商品を自動で候補登録。推奨購入日も自動付与 |
| 📅 購入日判断 | 5と0のつく日・5のつく日・スーパーセール/お買い物マラソン時期などのカレンダーから、最も実質仕入額を下げられる購入日を推奨 |
| 🏪 販路設定 | Amazon(FBA/自己発送)・メルカリ・ヤフオク!・ラクマ・Yahoo!フリマなどの手数料体系を自由に設定 |
| 📦 仕入管理 | 候補からワンクリックで仕入登録。数量・獲得ポイント・状態(在庫/出品中/売却済)を管理 |
| 💰 売上管理 | 売却登録時に販路手数料・ポイント按分原価を自動計算して純利益を算出 |
| 📈 レポート | 月次の売上・仕入・純利益・利益率、販路別実績、CSVエクスポート(Excel対応) |

## セットアップ

```bash
pip install -r requirements.txt
python app.py
```

ブラウザで http://127.0.0.1:5050 を開きます。
データは `sedori.db`(SQLite)に保存されます。

## APIキーの設定

初期状態では**デモデータ**で動作します。実データを検索するには「設定」画面で
APIキーを登録してください。

- **楽天市場**: [楽天ウェブサービス](https://webservice.rakuten.co.jp/) でアプリIDを取得(無料)
- **Yahoo!ショッピング**: [Yahoo!デベロッパーネットワーク](https://developer.yahoo.co.jp/) でClient IDを取得(無料)

## 使い方の流れ

1. **販路設定** — 自分が使う販路の手数料・送料を確認/調整
2. **設定** — APIキー、利益条件(最低利益額/利益率/ROI)、想定売価係数を設定
3. **商品リサーチ** — キーワード検索 or 保存キーワードの一括自動リサーチ
4. **候補商品** — 利益条件を満たした商品を確認し「仕入予定」へ
5. **購入日判断** — 推奨購入日(ポイントアップ日)を確認して仕入実行
6. **仕入管理 → 売上管理** — 仕入登録 → 売却登録で純利益を自動計算
7. **レポート** — 月次・販路別の実績を確認、CSVで出力

## 注意事項

- 想定売価は既定で「仕入価格 × 係数(初期値1.35)」の概算です。実運用では
  Keepa等で相場を確認し、候補登録時に想定売価を上書きしてください。
- セールイベントのうち「予測」表示のもの(お買い物マラソン等)は開催傾向からの
  推定です。必ず公式の開催情報を確認してください。
- 各モールのAPI利用規約・出品規約を遵守してください。
