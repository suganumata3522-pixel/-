# ナガシマスパーランド プール お誘いチラシ（ポケモン風デザイン）

小学生の友達をナガシマスパーランド「ジャンボ海水プール」に誘うためのチラシです。
まず日程を決めるため、**開催候補日（2026/7/21〜8/7 の平日）** を〇をつけて選んでもらう形式にしています。

## 成果物
- `flyer.png` … チラシ画像（A4縦 / 200dpi）
- `flyer.pdf` … 印刷用PDF（A4縦）

## 作り直し・編集方法
日付や金額、文言を変えたいときは `build_flyer.py` の該当箇所を編集して再生成します。

```bash
pip install Pillow
python3 build_flyer.py
```

`flyer.png` と `flyer.pdf` が再出力されます。

## 素材
- `assets/pool.jpg` … プールの写真（元のWord資料より）
- `assets/line_qr.jpg` … 連絡用LINEのQRコード（元のWord資料より）
- `assets/MPLUSRounded-*.ttf` … フォント「M PLUS Rounded 1c」(SIL Open Font License 1.1)

## デザインの方針
- ポケモン（でんきタイプ）風の配色：ピカチュウイエロー / モンスターボール赤 / ポケモンブルー
- モンスターボール・いなずま・ピカチュウ風マスコットの飾り
- 元のWord資料の内容（スケジュール・参加費・持ち物・連絡先）を踏襲

> ※ ポケモンは株式会社ポケモン等の商標です。本チラシは個人利用（友人へのお誘い）を想定した非公式のファンデザインです。
