# 層せん断力・各階地震力 算定問題集

ゼネコン構造設計部員の実務トレーニング用問題集。
建築基準法施行令第88条に基づき、許容応力度計算（一次設計）における地震力の算定を扱う。

## 構成

| ファイル | 内容 |
| --- | --- |
| `docs/00_formulas.md` | 基本式・記号・地盤種別等のリファレンス |
| `docs/01_basic.md` | 基礎問題（記号の定義、単純計算） |
| `docs/02_standard.md` | 標準問題（4〜6階建てモデル） |
| `docs/03_advanced.md` | 応用問題（混構造・特殊地盤・偏心） |
| `docs/04_solutions.md` | 解答・解説 |

## 使い方

1. `docs/00_formulas.md` で式を確認
2. `docs/01_basic.md` → `02_standard.md` → `03_advanced.md` の順に解く
3. `docs/04_solutions.md` で答え合わせ
