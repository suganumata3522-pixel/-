# Pikmin Throw (ファンメイド習作)

React Native (Expo) で作られたピクミン風モバイルシューティングゲームです。

> ⚠️ **権利について**：ピクミン / オリマーは任天堂株式会社の著作物です。本プロジェクトは個人の学習目的のファンメイド習作であり、配布・販売・ストア公開は行わないでください。

## ゲーム内容

- プレイヤーは画面下部の **オリマー風キャラ**
- 指で画面をドラッグして左右に移動
- 自動で **ピクミン（赤・黄・青がローテーション）** を投げる
- 上から降ってくる **チャッピー風の敵** をピクミンで倒す
- 敵に衝突するとライフが減り、3回で GAME OVER

## セットアップ

```bash
npm install
npm start
```

Expo Go アプリ（iOS / Android）で QR コードを読み取るとプレイできます。
ブラウザで試す場合は `npm run web` を実行してください。

## 技術スタック

- React Native 0.73
- Expo SDK 50
- TypeScript
- すべてのキャラは View シェイプで描画（外部画像なし）
