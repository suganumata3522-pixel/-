# 構造設計 実務トレーニング教材

ゼネコン構造設計部員向けの教材集。
新人〜若手向けの計算問題集と、中堅向けの判断・合意形成ワークショップで構成する。

## 構成

### 新人〜若手向け：計算問題集

建築基準法体系に沿った許容応力度計算・荷重算定の実務スキルを段階的に習得する。

### `docs/seismic/` 層せん断力・各階地震力（令第88条、Ai 分布）

| ファイル | 内容 |
| --- | --- |
| `00_formulas.md` | 基本式・記号・地盤種別等のリファレンス |
| `01_basic.md` | 基礎問題（記号定義、単純計算） |
| `02_standard.md` | 標準問題（4〜6階建てモデル） |
| `03_advanced.md` | 応用問題（混構造・特殊地盤・必要保有水平耐力） |
| `04_solutions.md` | 解答・解説 |

### `docs/wind/` 風圧力・ピーク風圧力（令第87条、平12告示1454/1458号）

| ファイル | 内容 |
| --- | --- |
| `00_formulas.md` | 基本式・記号・告示参照位置 |
| `01_basic.md` | 基礎問題（V0 定義・調査、骨組み用 vs 外装材用） |
| `02_standard.md` | 標準問題（粗度区分、Er・Gf・q、Cf、高さ方向分布） |
| `03_advanced.md` | 応用問題（ピーク風圧、隅角部、実務総合、地震との比較） |
| `04_solutions.md` | 解答・解説 |

### 中堅向け：判断・合意形成ワークショップ

実務経験を「応用する力・状況に応じ考える力・調整する力・根本を説明する力」へ発展させる、
正解のない課題のロールプレイ形式。

#### `docs/ve_workshop/` VE・合意形成（躯体コスト1割減）

| ファイル | 内容 | 配布先 |
| --- | --- | --- |
| `00_guide.md` | 狙い・進め方・タイムテーブル・評価軸 | 全員＋進行役 |
| `01_case.md` | 共通与条件（プロジェクト・構造・コスト・VE要求） | 全員 |
| `02_roles.md` | 役割カード（公開＋非公開の利害） | **役ごとに切分けて配布** |
| `03_tasks.md` | 課題（個人ワーク・調整会議の進め方・論点） | 全員 |
| `04_solutions.md` | 解説（VEメニュー・模範合意解・講評・ルーブリック） | **進行役のみ** |

## 使い方

### 計算問題集（seismic / wind）

1. `00_formulas.md` で式と告示の引き方を確認
2. `01_basic.md` → `02_standard.md` → `03_advanced.md` の順に解く
3. `04_solutions.md` で答え合わせ
4. 風問題集は **平12建告1454号・1458号を手元に置いて** 解くこと（数値は告示参照）

### ワークショップ（ve_workshop）

1. 進行役が `00_guide.md` で狙いと流れを把握
2. `01_case.md` を全員に配布、`03_tasks.md` 課題①を個人ワーク
3. `02_roles.md` を**役ごとに切り分けて**配布し、調整会議を実施
4. `04_solutions.md` で講評（会議前は進行役のみ閲覧）
