# Pikmin Throw (ファンメイド習作)

React Native (Expo) で作られたピクミン風モバイルシューティングゲームです。

## ゲーム内容

- プレイヤーは画面下部の **オリマー風キャラ**
- 指で画面をなぞった方向にピクミン（赤・黄・青）を投げる
- 触れていない時はまっすぐ上に投げる
- ピクミンは敵に**張り付いて**ダメージを与える（離れない）
- 張り付くピクミンが多いほど早く倒せる（群がるほど強い）
- 敵 HP は 5。倒すと **30点 + 張り付き数×10点** がスコアに加算
- 敵に衝突するとライフが減り、3回で GAME OVER

## 操作

| 操作 | 効果 |
|------|------|
| 画面をタップ・ドラッグ | オリマーが指のX座標に追従＋指の位置に向けて投げる |
| 何もしない | まっすぐ上に投げる |

## セットアップ

```bash
npm install
npm start
```

Expo Go アプリ（iOS / Android）で QR コードを読み取るとプレイできます。
ブラウザで試す場合は `npm run web` を実行してください。

## 技術スタック

- React Native 0.73
- Expo SDK 50
- TypeScript
- すべてのキャラは View シェイプで描画（外部画像なし）
