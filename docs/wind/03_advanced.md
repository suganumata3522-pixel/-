## 問題 3-2 （カーテンウォール隅角部のピーク風圧力 ― 中層と超高層の比較）

カーテンウォールは中層〜超高層オフィスで採用される外装材。中層と超高層で wp がどう変わるかを比較する。

### 共通条件
- 建設地：東京都港区、地表面粗度区分 IV、V0 は告示より引く
- 閉鎖型建物、Ĉpi = +0.5（最不利、外向き吹き飛ばし最大）
- Ĉpe（告示1458号 代表値）：
   - 壁面 一般部 Ĉpe = −1.0
   - 壁面 周辺部 Ĉpe = −1.8
   - 壁面 隅角部 Ĉpe = −3.0

### 3-2 (a) 中層オフィスビル（H = 40 m）

(1) q(H=40, 粗度IV) を求めよ
(2) 各部位（一般・周辺・隅角）の **wp** を算出（kN/m²）
(3) 一般部 / 周辺部 / 隅角部 の比をとり、隅角部のファスナーを別仕様にする必要性を説明

### 3-2 (b) 超高層オフィスビル（H = 100 m）

(1) q(H=100, 粗度IV) を求めよ
(2) 各部位の wp を算出
(3) **(a) の中層と (b) の超高層で隅角部 wp を比較**し、高さスケールが外装材設計に与える影響を述べよ

### 3-2 (c) 考察

- 中層と超高層で **wp の差** は q の差に比例する（Ĉpe は同じ）。
- 高層化に伴うコスト：q 上昇分のファスナー強度・板厚・専用工事
- なぜ低層（H<10m）ではこの議論があまり問題化しないかを述べよ（ヒント：q が小さく葺き材標準仕様で足りる）

## 問題 3-4 （内圧の最不利組合せ）

(1) 同じ建物について、Ĉpi を **0**（開口少）と **−0.5**（風下側のみ開口）で取り得ると仮定する。
   壁面 隅角部の Ĉpe = −2.5（負圧）の場合、最不利の wp はどう選ぶか。
   計算過程を示せ。

(2) 風上面の Ĉpe = +1.5（正圧）の場合、最不利の Ĉpi はどちらを取るか。
   理由とともに答えよ。

(3) 一般則として、外装材設計では「**Ĉpe と Ĉpi の差を最大化する組合せ**」を選ぶことを覚える。
   - 押し（内向き、面が室内に押される）：Ĉpe(+) − Ĉpi(−)
   - 引き（外向き、面が外に引っ張られる）：Ĉpe(−) − Ĉpi(+)

## 問題 3-5 （実務総合：10階建てオフィスビル）

### 条件
- 構造：S造、10階建て、H = 40 m（階高 4.0m × 10）
- 建設地：横浜市西区、地表面粗度区分 III
- 平面：X 方向 40m × Y 方向 25m、長辺平行が X
- V0 は告示より引く（横浜）
- 検討する風向：X 方向風（短辺風、風上面 25m × 40m）

### 設問

(1) **V0 調査**：横浜市西区の V0 を告示より引け

(2) **q の高さ方向分布**：高さを 4 分割し、各層中央で q を評価せよ
   | 層 | 高さ範囲 (m) | 評価高さ (m) | Er | Gf | E | q (N/m²) |
   | --- | --- | --- | --- | --- | --- | --- |
   | A | 30〜40 | 35 | | | | |
   | B | 20〜30 | 25 | | | | |
   | C | 10〜20 | 15 | | | | |
   | D | 0〜10  | 10（最小高さ）| | | | |
   
   ※ H ≤ Zb = 5m の部分は H = Zb で評価（Er は H=Zb 値で頭打ち）

(3) **風力係数 Cf**：閉鎖型建物として
   - 風上面：Cpe = +0.8、Cpi = −0.2（最不利）⇒ Cf = +1.0
   - 風下面：Cpe = −0.4、Cpi = +0.2（最不利）⇒ Cf = −0.6
   - 全押し引き合算：1.0 + 0.6 = 1.6（簡略のため両面合計を風上面に作用と扱う）

(4) **各層の風荷重**：各層に作用する水平力 Pi を求めよ
   - 受圧面積 Ai = 25m × 10m = 250 m²
   - Pi = q(中央高さ) · Cf · Ai （Cf = 1.6 を使用）

(5) **ベース剪断力 Qb**：Qb = ΣPi を求めよ

(6) **転倒モーメント Mb**：各層 Pi × 評価高さ の合計を求めよ（基礎位置 z=0 まわり）

## 問題 3-7 （考察）

(1) 同じ建物を **粗度区分 II と IV** に置いた場合、低層部（H=10m）の q はどちらが大きいか。
   告示 Gf 表を見ると、粗度IV のほうが H=10m で Gf が大きい一方、Er は粗度IV のほうが小さい。
   両者の積 E = Er²·Gf を比較して、定性的に答えよ。

(2) 設計用 V0 を保守的に＋10% で見ると、q は何 % 増加するか？（q ∝ V0² から）
   設計風荷重への影響を述べよ。

(3) 粗度区分を **危険側（実際よりラフ寄りに）誤った場合**、低層の q は過大か過小か？
   ⇒ 安全側か危険側かを判定せよ。
