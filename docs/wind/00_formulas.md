# 00. 風荷重 基本式・記号リファレンス

ゼネコン構造設計部・新人向け。
建築基準法施行令第87条、平12建告1454号（風圧力）、平12建告1458号（外装材ピーク風圧）に基づく。

数値表（V0、Er の Zb/ZG/α、Gf、Cf、ピーク係数 等）は**告示本文を手元に置いて引く**こと。
本書では式と引き方のみ示す。

## 1. 体系図

```
┌─ 構造骨組み用 ────────────┐    ┌─ 外装材用 ───────────┐
│ W = q · Cf       [N/m²]   │    │ wp = q · Ĉf  [N/m²]   │
│  Cf = Cpe − Cpi           │    │  Ĉf = Ĉpe − Ĉpi       │
│  （ガスト Gf は q に含む）│    │  （ピーク係数で局所突風考慮）│
└───────────────────────────┘    └───────────────────────┘
```

| 用途 | 算定式 | 根拠 | 適用 |
| --- | --- | --- | --- |
| 構造骨組み用 | `W = q · Cf` | 令87条、告示1454号 | 柱・梁・耐震壁・基礎の応力算定 |
| 外装材用 | `wp = q · Ĉf` | 告示1458号 | カーテンウォール・ガラス・葺き材・間柱 |

## 2. 平均速度圧 q

```
q = 0.6 · E · V0²   [N/m²]
E = Er² · Gf
```

| 記号 | 意味 | 引き先 |
| --- | --- | --- |
| V0 | 基準風速 (m/s) | 平12告1454号 第2（市町村別） |
| Er | 平均風速の高さ方向分布係数 | 平12告1454号 第1（粗度区分・高さ） |
| Gf | ガスト影響係数 | 平12告1454号 別表第3（粗度・高さ） |
| E  | Er²·Gf（環境係数） | 計算で求める |

## 3. 基準風速 V0

- **定義**：再現期間50年・10分間平均風速・地上10m・地表面粗度区分II相当
- **範囲**：30〜46 m/s（地域による）
- **引き方**：平12告1454号 第2の表で都道府県 → 市町村単位で確定
- 地方公共団体の指定で特例（沿岸・離島・峡谷等）あり

## 4. 地表面粗度区分 (告示1454号 第1)

| 区分 | 内容 |
| --- | --- |
| I   | 都市計画区域外で極めて平坦・障害物が少ない（沖合・草地） |
| II  | 田園地帯、樹木や低層建築物が散在（**標準**） |
| III | 樹木・低層建物が多い、または中層が散在 |
| IV  | 大都市の中心部（中高層が密集） |

各区分に対し `Zb`, `ZG`, `α` が定義される（告示参照）。

**Er の計算**：

| 高さ範囲 | 式 |
| --- | --- |
| `H ≤ Zb` | `Er = 1.7 · (Zb / ZG)^α` |
| `H >  Zb` | `Er = 1.7 · (H  / ZG)^α` |

## 5. ガスト影響係数 Gf

- 平12告1454号 別表第3 より、**粗度区分 × 高さ H** で引く
- H = 10m 以下、40m、≥ 100m の3点が表に与えられ、間は直線補間
- 粗度が粗いほど、また低層ほど Gf は大きい

## 6. 風力係数 Cf （構造骨組み用、告示1454号 別表第1〜2）

閉鎖型建物：

```
Cf = Cpe − Cpi
```

| 面 | Cpe（外圧係数） |
| --- | --- |
| 風上面（壁） | 正圧 `+0.8·kz`（kz：高さ方向の補正） |
| 風下面（壁） | 負圧 −0.4 程度（H/D 比による） |
| 側面（壁） | 負圧 −0.7 程度 |
| 屋根（陸屋根・勾配付き） | 部位別に告示参照 |

| Cpi（内圧係数） | 適用 |
| --- | --- |
|  0 または +0.2 | 開口が少ない閉鎖型（最不利を選択） |
| −0.2 | 同上 負側 |
| 大開口あり | 別途、告示の対応値 |

設計では `Cpe` と `Cpi` の最不利組合せ（差を最大化する組合せ）を用いる。

## 7. ピーク風圧力（外装材用、告示1458号）

```
wp = q · ( Ĉpe − Ĉpi )   [N/m²]
```

- `Ĉpe`：ピーク外圧係数。同じ面でも **隅角部 > 周辺部 > 一般部** で絶対値が大きい
- `Ĉpi`：ピーク内圧係数。閉鎖型なら 0 / −0.5 程度の最不利組合せ
- **理由**：外装材は小面積に瞬間突風が局所的に作用する。骨組み用 W は時間・空間平均で評価しており、外装材には不足。

## 8. 計算手順（実務フロー：骨組み用）

1. 建設地の **V0** を告示1454号より引く
2. 建設地周辺状況から **粗度区分** を判定（条例・指定区域も確認）
3. 設計高さ H に応じ **Er** を計算（Zb・ZG・α は告示）
4. H・粗度区分から **Gf** を表で引く（間は補間）
5. `E = Er² · Gf` を計算
6. `q = 0.6 · E · V0²` を計算
7. 建物形状から **Cpe・Cpi** を引き、`Cf = Cpe − Cpi`
8. `W = q · Cf` を面・部位別に算定
9. （高層・長スパン等）必要に応じ高さ方向に q を段階分けして W(z)

## 9. よくあるミス

- V0 は **市町村単位**で異なる。県全体で一律ではない
- 粗度区分 II が「標準」だが、**実際の建設地は III または IV** のことが多い
- Er は粗度ごとに `Zb`・`ZG`・`α` が異なる → 表の読み間違いに注意
- Gf は H で連続値（補間必要）。表の3点だけでない
- **骨組み用 W と外装材用 wp は別物**。同じ建物でも数値が異なる
- 外装材設計を Gf で済ませると不足（隅角部のピーク突風が再現できない）
- 単位は N/m²（kN/m² ではない）。kN/m² に換算するときは 1/1000
