# 00. 基本式・記号リファレンス

## 1. 地震層せん断力係数 Ci

```
Ci = Z · Rt · Ai · C0
```

| 記号 | 意味 | 一次設計 |
| --- | --- | --- |
| `Z`  | 地域係数 | 0.7 〜 1.0（地域による） |
| `Rt` | 振動特性係数 | 建物周期 T と地盤種別により決定 |
| `Ai` | 高さ方向の分布係数 | i 層位置と T により決定 |
| `C0` | 標準せん断力係数 | 一次設計：0.2（必要保有水平耐力時：1.0以上） |

## 2. 設計用一次固有周期 T

```
T = h · (0.02 + 0.01 · α)
```

- `h`：建築物の高さ（m）
- `α`：鉄骨造部分の高さの割合（0〜1）
  - 純RC造・純SRC造： α = 0 → `T = 0.02 h`
  - 純S造（鉄骨造）： α = 1 → `T = 0.03 h`

## 3. 振動特性係数 Rt

地盤の卓越周期 Tc を用いて区分計算する。

| 範囲 | Rt |
| --- | --- |
| `T < Tc`              | `Rt = 1.0` |
| `Tc ≤ T < 2 Tc`       | `Rt = 1 − 0.2 · (T/Tc − 1)²` |
| `T ≥ 2 Tc`            | `Rt = 1.6 · Tc / T` |

地盤種別と Tc：

| 地盤種別 | 内容 | Tc (秒) |
| --- | --- | --- |
| 第1種 | 硬質地盤（岩盤・洪積層等） | 0.4 |
| 第2種 | 普通地盤（第1・第3種以外） | 0.6 |
| 第3種 | 軟弱地盤（沖積層・埋立地等） | 0.8 |

## 4. Ai 分布

```
Ai = 1 + ( 1/√αi − αi ) · ( 2T / (1 + 3T) )
```

- `αi = Wi / W`
  - `Wi`：i 層より上の重量の合計（i 層が支える重量）
  - `W` ：建物全体の重量（地上部）
- 最上層で `αi = Wn/W`、最下層（1階）で `αi = 1` ⇒ `A1 = 1`

## 5. 層せん断力 Qi

i 層が負担するせん断力：

```
Qi = Ci · Wi
```

`Wi` は i 層より上の重量の合計（= αi · W）。

## 6. 各階に作用する水平地震力 Pi

```
Pi = Qi − Q(i+1)
```

最上層では `P_n = Q_n`（上に層がないため）。
※ Pi は i 階の床位置に水平に集中させる地震力として扱う。

## 7. 計算手順（実務フロー）

1. 各階の重量 Wj を算出（固定荷重 + 積載荷重 地震時 + 多雪区域では雪荷重）
2. 全体重量 W と Wi（各層より上の合計）を算出
3. 建物高さ h と材料区分から T を計算
4. 地盤種別から Tc を決定し Rt を計算
5. αi = Wi/W を求め、各層の Ai を計算
6. Ci = Z · Rt · Ai · C0 を計算
7. Qi = Ci · Wi を計算
8. Pi = Qi − Q(i+1) を計算

## 8. よくあるミス

- `Wi` は「i 階の重量」ではなく「i 層より上の重量の合計」
- 最上層の `Ai` が最大、1 層が `A1 = 1.0`
- `Ci` も同様に最上層が最大、1 層が最小ではないので注意（`Ai` 増加 vs `Wi` 減少のため一概に言えない）
- `Qi` は単調増加（下層ほど大）
- T の計算は m 単位（mm 不可）
- 地下部分の地震力は別途 `k = 0.1 · Z · (1 − H/40)` で計算（本問題集では地上部のみ扱う）
