# ⚾ ニュースの調べ方マニュアル

野球ニュースを自動で集める操作の手順書です。
パソコンの操作に慣れていない方でも、この通りにやれば必ず動くように書いています。

> 📌 このマニュアルでできること
> **インターネット上の野球ニュースを 30 件ほど自動で集めて、読みやすい一覧にする**
> （AI で台本を作る機能は別の有料機能です。このマニュアルには含みません）

---

## 🟢 毎回やること（3ステップ・3分）

ニュースを集めたくなったら、毎回この 3 ステップをやるだけです。

### ステップ 1️⃣ プログラムのフォルダで「PowerShell」を開く

1. **エクスプローラー**（フォルダを開くアプリ）を起動する
   - キーボードの **Windows キー ＋ E** で開きます
2. 次の場所を開く（プログラム本体を置いたフォルダ）

   ```
   ドキュメント ＞ baseball-news
   ```

   ※ `main.py` というファイルが直接見える場所です
3. 画面上の方の **アドレス欄**（`... ＞ baseball-news` と書いてある細長いバー）を **クリック**
4. そこに **`powershell`** と打って **Enter** キー
5. 青っぽい黒い画面（PowerShell）が開きます

> 💡 **このテクニックが一番大事**です。
> 「そのフォルダで PowerShell を開く」＝ **アドレス欄に `powershell` と打つ**、と覚えてください。
> 画面の左端に `... \baseball-news>` と表示されていれば正しい場所です。

### ステップ 2️⃣ コマンドを 1 行打つ

開いた黒い画面に、次の文字をコピーして貼り付け、**Enter** を押します。

```
py main.py collect
```

- 文字がたくさん流れます（インターネットからニュースを読み込み中）
- **1〜3 分くらい** かかります。途中で止まって見えても、あわてず待ってください

うまくいくと、最後のあたりにこう表示されます 👇

```
[collect] 統合後の話題数: 30
[collect] 保存: out\topics_20260602_223613.json
[collect] 読みやすい版: out\topics_20260602_223613.txt
[collect] 上位 10 件:
  ...（ニュースのタイトルがいくつか表示される）
```

### ステップ 3️⃣ 集めた結果を読む

1. エクスプローラーで `baseball-news ＞ out` フォルダを開く
2. **`topics_latest.txt`** というファイルを **ダブルクリック**
3. メモ帳が開いて、ニュース一覧が読みやすく表示されます ✨

これで完了です！🎉

---

## 📁 out フォルダにできるファイルの意味

`py main.py collect` を実行すると、`out` フォルダに 4 つのファイルができます。

| ファイル名 | 中身 | 人が読む用？ |
|---|---|---|
| **`topics_latest.txt`** | 👀 **これを開けばOK**。最新の結果を読みやすくしたもの | ✅ 読む用 |
| `topics_latest.json` | 最新の結果（機械が読む形式） | ⚙️ AI台本用 |
| `topics_20260602_223613.txt` | 上の txt の「日時つき保存版」 | ✅ 過去分の記録 |
| `topics_20260602_223613.json` | 上の json の「日時つき保存版」 | ⚙️ 過去分の記録 |

- **普段は `topics_latest.txt` だけ見ればOK**（毎回いちばん新しい結果に更新されます）
- 数字（`20260602_223613`）は「2026年6月2日 22時36分13秒」という意味の日時です
- 過去の結果も残るので、後から見返せます

> ⚠️ **JSON ファイルは編集しないでください。**
> 将来 AI で台本を作るときの「材料データ」です。見るだけにしてください。
> もし開いて保存を聞かれても、必ず「保存しない／キャンセル」を選んでください。

## 📖 一覧（txt）の読み方

`topics_latest.txt` を開くと、こんな形でニュースが並びます 👇

```
【 1】 score  13.28   出典: full-count.jp
  タイトル: 米国で覚醒した元NPB右腕…識者が絶賛「圧倒的に最高」
  要約    : レイズのニック・マルティネス投手が今季、驚異的な…
  公開    : 2026-06-02 20:50
  URL     : https://full-count.jp/2026/06/02/post1971419/
  ID      : 1c2c454940d1  ← 台本生成で使う ID
----------------------------------------------------------------------
```

各項目の意味:

| 項目 | 意味 |
|---|---|
| **【番号】** | 注目度ランキングの順位（1 が一番注目） |
| **score** | 注目度・拡散度の点数。**高いほど話題**になっているニュース |
| **出典** | どのサイトのニュースか |
| **タイトル** | ニュースの見出し |
| **要約** | 内容の短いまとめ |
| **公開** | ニュースが出た日時（日本時間） |
| **URL** | 元記事のアドレス。コピーしてブラウザで開けば全文が読めます |
| **ID** | このニュースの識別番号（将来 AI 台本を作るときに使う） |

## 🆘 困ったときは

### ❓「python は認識されません」と赤い字が出る
`python` ではなく **`py`** で打ってください（2文字）。
- ❌ `python main.py collect`
- ✅ `py main.py collect`

### ❓「main.py が見つかりません」のようなエラー
PowerShell を開いたフォルダが間違っています。
1. いったん黒い画面を閉じる
2. **ステップ 1** をやり直し（`main.py` が直接見えるフォルダで `powershell`）
3. 確認コマンド `dir main.py` を打って、ファイル情報が出ればOK

### ❓ 集まった件数が「0」だった
- インターネットに繋がっているか確認
- 少し時間をおいて、もう一度 `py main.py collect` を実行
- それでも 0 なら、画面の写真を撮って相談してください

### ❓ 赤い字で「ERROR」がたくさん出た
1 つのニュース源が一時的に落ちていても、他は集まるので多くは問題ありません。
最後に「統合後の話題数: ○○」と出ていれば成功しています。
件数が 0 で本当に止まる場合だけ、画面の写真を撮って相談してください。

### ❓ 黄色い字の「WARNING」が出た
**無視してOK**です。「念のためのお知らせ」で、エラーではありません。

## 🔄 どれくらいの頻度でやればいい？

- ニュースは時間とともに増えるので、**動画を作りたいタイミングで毎回** 実行するのがおすすめ
- 同じ日に何回やってもOK（そのたびに最新の結果に更新されます）
- 過去の結果は日時つきファイルで残るので、消えません

## 📝 用語ミニ辞典

| 言葉 | かんたんな意味 |
|---|---|
| **PowerShell** | コマンド（命令文）を打ってプログラムを動かす「黒い画面」 |
| **Python（パイソン）** | このプログラムを動かす土台ソフト。`py` で呼び出す |
| **コマンド** | パソコンへの命令文。今回は `py main.py collect` の1つだけ覚えればOK |
| **JSON（ジェイソン）** | 機械が読むためのデータ形式。記号が多くて人には読みにくい |
| **out フォルダ** | 集めた結果が保存される場所 |
| **score（スコア）** | ニュースの注目度の点数 |

## 📌 （参考）新しいパソコンで一から始める場合

別のパソコンで使うときは、最初に一度だけ次の準備が必要です。
（普段の利用では不要です）

1. **Python を入れる** … <https://www.python.org/downloads/> から入れる
   - PowerShell で `py --version` と打って数字が出ればOK
2. **プログラム本体を置く** … GitHub から ZIP をダウンロードして `baseball-news` フォルダに展開
   - `main.py` が直接見える状態にする
3. **部品を入れる** … `baseball-news` フォルダで PowerShell を開き、次を 1 回実行

   ```
   py -m pip install -r requirements.txt
   ```
   最後に `Successfully installed ...` と出ればOK

準備ができたら、あとは上の「**毎回やること（3ステップ）**」を繰り返すだけです。
