# 電脳せどり管理ツール かんたん操作マニュアル

はじめて使う方向けに、実際の画面を見ながら操作の流れを説明します。

---

## 1. このツールでできること

「ネットで安く仕入れて、別の場所で高く売る(電脳せどり)」の作業を、ぜんぶ1つの画面で管理できます。

| やりたいこと | 使う画面 |
| --- | --- |
| 儲かる商品を探したい | 商品リサーチ |
| 探すのを自動化したい | 商品リサーチ(自動リサーチ) |
| いつ買うのが一番お得か知りたい | 購入日判断 |
| どこで売るかの手数料を管理したい | 販路設定 |
| 買ったもの・売れたものを記録したい | 仕入管理・売上管理 |
| 今月いくら儲かったか見たい | ダッシュボード・レポート |

### 基本の流れ

```
 ① リサーチで商品を探す
       ↓
 ② 良さそうな商品を「候補」に登録
       ↓
 ③ 「購入日判断」でお得な日に仕入れる → 仕入登録
       ↓
 ④ 売れたら売上登録(利益は自動計算)
       ↓
 ⑤ レポートで振り返り
```

## 2. はじめる(起動方法)

このツールはお使いのパソコンの中で動く小さなアプリです。
最初に一度だけ準備(①と②)をすれば、**2回目からはファイルをダブルクリックするだけ**で使えます。

### 準備① Python を入れる(最初の1回だけ)

このツールは「Python(パイソン)」という無料ソフトの上で動きます。

1. ブラウザで [python.org/downloads](https://www.python.org/downloads/) を開く
2. 黄色い「**Download Python 3.x.x**」ボタンを押してダウンロード
3. ダウンロードしたファイルをダブルクリックしてインストール
   - **Windowsの方**: 最初の画面の下にある
     「**Add python.exe to PATH**」に**必ずチェック**を入れてから「Install Now」
     (これを忘れると起動できません。忘れた場合は入れ直せばOKです)
   - **Macの方**: そのまま「続ける」で進めればOK

### 準備② ツール一式をパソコンに置く(最初の1回だけ)

1. このツールのGitHubページで、緑色の「**Code**」ボタン →「**Download ZIP**」を押す
2. ダウンロードしたZIPを右クリック →「すべて展開」(Macはダブルクリック)
3. できたフォルダを、デスクトップなどわかりやすい場所に置く

### 起動する(毎回これだけ)

フォルダの中の、お使いのパソコンに合ったファイルを**ダブルクリック**します。

| パソコン | ダブルクリックするファイル |
| --- | --- |
| Windows | `start_windows.bat` |
| Mac | `start_mac.command` |

すると——

1. 黒い画面(ターミナル)が開きます。**初回だけ**自動で部品のインストールが走ります(1〜2分)
2. 続けて**ブラウザが自動で開き**、ツールの画面が表示されます
3. 開かない場合は、ブラウザに `http://127.0.0.1:5050` と入力してください

> ⚠️ **使っている間、黒い画面は閉じないでください**(最小化はOK)。
> 黒い画面=ツール本体です。終わるときはこの画面を閉じるだけ。
> 入力したデータはフォルダ内の `sedori.db` に保存されているので、次回も続きから使えます。

> 💡 Macで「開発元が未確認のため開けません」と出たら、
> ファイルを**右クリック →「開く」→「開く」**を選んでください(初回のみ)。

### (参考)コマンドで起動する場合

パソコン操作に慣れている方は、フォルダでターミナルを開いて次の2行でも起動できます。

```bash
pip install -r requirements.txt
python app.py
```

## 4. 最初にやる設定(5分)

### 4-1. 販路設定 — 「どこで売るか」の手数料を確認

メニューの「販路設定」を開きます。Amazon・メルカリ・ヤフオク!などが
最初から登録済みなので、**自分が使う販路の数字を確認するだけでOK**です。

![販路設定](images/02_channels.png)

- **手数料率(%)** … 売れたときに販路に取られる手数料(例: メルカリは10%)
- **固定手数料(円)** … 1個売るごとにかかる固定費(例: FBAの出荷費用)
- **標準送料(円)** … 発送にかかる送料のめやす

> 💡 ここの数字を使って、リサーチ時の利益が自動計算されます。

### 4-2. 設定 — APIキーと利益の条件

メニューの「設定」を開きます。

![設定](images/03_settings.png)

**① 仕入元API(無料)**
- 楽天アプリID … [楽天ウェブサービス](https://webservice.rakuten.co.jp/)で無料発行
- Yahoo! Client ID … [Yahoo!デベロッパーネットワーク](https://developer.yahoo.co.jp/)で無料発行

> ⚠️ 未設定でも「デモデータ」で全機能を試せます。まず触ってみて、慣れたらAPIキーを入れましょう。

**② Amazon売価の自動取得(任意)**
- Keepa APIキー(有料)または Amazon SP-API の認証情報を入れると、
  商品のJANコードから**Amazonでの実際の販売価格**を自動で取ってきて、想定売価に使います。
- どちらも未設定の場合は「仕入価格×係数」のざっくり推定になります。

**③ 利益商品の判定条件**
- 「利益500円以上・利益率10%以上・ROI 15%以上」が初期値。
  自動リサーチはこの条件を満たした商品だけを拾います。お好みで変更してください。

## 5. 商品リサーチ(手動で探す)

メニューの「商品リサーチ」でキーワードを入れて「検索」を押すと、
仕入候補が利益計算つきで一覧表示されます。

![商品リサーチ](images/04_research.png)

### 結果の見方

| 列 | 意味 |
| --- | --- |
| 仕入価格 | 今その商品を買える値段 |
| pt還元 | 仕入時にもらえるポイント(実質値引きとして計算) |
| 想定売価 | 売れるであろう値段。<span style="color:#065f46">**「Amazon相場」**</span>バッジ付きはAmazonの実勢価格、「係数推定」はざっくり計算 |
| 利益 | 売価 − 手数料 − 送料 − 実質仕入額。**緑の行は利益条件クリア** |
| 利益率 | 利益 ÷ 売価 |
| ROI | 利益 ÷ 実質仕入額(投資効率。高いほど資金効率が良い) |

気になる商品は右端の **「候補登録」** を押すだけ。隣の数字ボックスで想定売価を
直してから登録することもできます(係数推定の商品は相場を調べて直すのがおすすめ)。

> 💡 JANコードがわかっている商品は、検索フォーム下の
> 「JANコードからAmazon相場を単体検索」でAmazon価格だけサッと調べられます。

---

## 6. 自動リサーチ(探すのを自動化)

リサーチ画面の下のほうにある「自動リサーチ(保存キーワード)」を使います。

![自動リサーチ](images/04b_research_auto.png)

1. よく探すキーワード(例「ゲームソフト 限定版」)と仕入上限額を登録
2. **「▶ 自動リサーチを一括実行」** を押す
3. 登録した全キーワードを一括検索し、**利益条件を満たした商品だけ**が
   自動で「候補商品」に登録されます(おすすめ購入日も自動でつきます)

> 💡 朝イチでボタンを1回押すだけで、その日の利益商品リストができあがります。

## 7. 候補商品を管理する

メニューの「候補商品」に、登録した商品が利益の大きい順に並びます。

![候補商品](images/05_candidates.png)

- 状態は **候補 → 仕入予定 → 仕入済**(やめるときは**見送り**)と進めます
- 「Amazon相場」バッジの商品は、ASINのリンクからAmazonの商品ページを直接確認できます
- **推奨購入日**列に「いつ買うのがお得か」とその理由が表示されます
- 買うと決めたら **「仕入登録」** ボタンへ(次の章)

---

## 8. 購入日を決める

メニューの「購入日判断」を開くと、今後30日間のセール・ポイントアップ日カレンダーと、
購入待ち商品の推奨購入日が見られます。

![購入日判断](images/06_timing.png)

- <span style="color:#065f46">**確定**</span> … 毎月決まっているルール(楽天「5と0のつく日」、Yahoo!「5のつく日」など)
- <span style="color:#6b21a8">**予測**</span> … 開催されることが多い時期(お買い物マラソン、スーパーセールなど)。
  **必ず公式の開催告知を確認してから**買ってください
- 黄色の行がおすすめ日です

> 💡 例えば楽天で15,000円仕入れる場合、+6倍の日に買うと約900円分お得になります。
> 利益500円の商品なら、買う日を変えるだけで利益がほぼ倍になる計算です。

---

## 9. 仕入を登録する

候補商品の「仕入登録」を押すと、商品名・価格・ポイントが**自動入力された状態**で
フォームが開きます。数量と日付を確認して「登録する」だけです。

![仕入登録](images/07_purchase_form.png)

登録した仕入は「仕入管理」に一覧表示されます。

![仕入管理](images/08_purchases.png)

- 状態は **在庫 → 出品中 → 売却済** で管理(売却済は売上登録時に自動で変わります)
- 「何個中何個売れたか」も表示されます
- 「CSVエクスポート」でExcelに出力できます(確定申告の資料づくりに便利)

---

## 10. 売れたら売上を登録する

仕入管理の **「売却登録」** を押すと、商品名などが入った状態でフォームが開きます。
**売れた値段と日付を入れるだけ**で、あとは全部自動計算です。

![売上登録](images/09_sale_form.png)

自動で計算されるもの:
- 販路の手数料(販路設定の手数料率・固定費を使用)
- 仕入原価(仕入時にもらったポイント分は割引いて計算)
- **純利益**

![売上管理](images/10_sales.png)

## 11. レポートで振り返る

メニューの「レポート」で、月ごとの売上・仕入・純利益・利益率と、
販路別の成績が確認できます。

![月次レポート](images/11_report.png)

「どの販路が一番儲かっているか」「先月より利益率が上がったか」をチェックして、
仕入の方針に活かしましょう。

## 12. 用語集

| 用語 | 意味 |
| --- | --- |
| 利益 | 売価 −(手数料+送料+経費)− 実質仕入額 |
| 実質仕入額 | 仕入価格からポイント還元分を引いた、本当の支出額 |
| 利益率 | 利益 ÷ 売価。20%以上なら優秀といわれます |
| ROI | 利益 ÷ 実質仕入額。「使ったお金が何%増えて返ってくるか」 |
| 想定売価 | 売れるであろう値段。Amazon相場(実勢価格)か係数推定 |
| カート価格 | Amazonで「カートに入れる」ボタンに表示される価格 |
| 売れ筋ランキング | Amazonのカテゴリ内順位。小さいほどよく売れている |
| JANコード | 商品バーコードの13桁の番号。商品の特定に使う |
| 買い回り | 楽天のセールで複数ショップ購入するとポイント倍率が上がる仕組み |

## 13. よくある質問

**Q. 起動用ファイルをダブルクリックしても、黒い画面が一瞬で閉じてしまいます**
A. ほとんどの場合、Pythonインストール時に「Add python.exe to PATH」の
チェックを忘れたのが原因です。python.orgからもう一度インストールし直して、
チェックを入れてから「Install Now」を押してください。

**Q. 間違えて黒い画面を閉じてしまいました**
A. ツールが止まるだけで、データは消えません。
もう一度起動用ファイルをダブルクリックすれば続きから使えます。

**Q. APIキーがなくても使えますか?**
A. 使えます。デモデータで全機能を体験できます。実際の商品を検索するには
楽天/Yahoo!の無料APIキーを「設定」画面に入れてください。

**Q. 「係数推定」の想定売価は信用していいですか?**
A. あくまでざっくりの目安です。Keepa等でAmazon相場を確認するか、
候補登録時に想定売価を実際の相場に直してください。
Keepa APIキーかSP-APIを設定すれば自動でAmazon実勢価格になります。

**Q. 「予測」と出ているセールは確実に開催されますか?**
A. いいえ。お買い物マラソン等は開催時期の傾向からの推定です。
購入前に必ず楽天・Yahoo!の公式告知を確認してください。

**Q. データはどこに保存されますか?**
A. ツールと同じフォルダの `sedori.db` ファイルです。
このファイルをコピーすればバックアップになります。

**Q. 複数個仕入れて、別々の日に売れた場合は?**
A. 売れるたびに売上登録すればOKです。ポイントや原価は個数分だけ
自動で按分され、全部売れたら仕入が自動で「売却済」になります。

**Q. スマホから使えますか?**
A. 同じWi-Fi内なら、`app.py` の `host` を `0.0.0.0` に変えて起動すると
スマホのブラウザからパソコンのIPアドレスでアクセスできます。
