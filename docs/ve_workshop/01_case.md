# 共通与条件 ― (仮)○○事務所ビル新築工事

全員が共有する前提。数値は訓練用の概算であり、実プロジェクトの精算値ではない。

## 1. プロジェクト概要

| 項目 | 内容 |
| --- | --- |
| 発注方式 | 設計施工（自社設計部が構造担当） |
| 用途 | 事務所（基準階）＋ 1〜2階エントランス・店舗 |
| 規模 | 地上6階・塔屋1階、延床 約 12,000 m² |
| 構造種別 | 鉄骨造（S造）、純ラーメン＋コア部に耐震ブレース |
| 基準階 | 36m × 36m、グリッド 7.2m × 7.2m |
| 正面アトリウム | **18m × 18m 無柱**、1〜2階 2層吹抜（天井高 約12m） |
| 基礎 | 場所打ちコンクリート杭、支持層 GL−22m |
| 工期 | 着工〜竣工 **22ヶ月（テナント入居日確定、延伸不可）** |

## 2. 構造設計の概要（当初案）

- **架構**：X・Y 両方向 純ラーメン。コア（EV・階段）周りに耐震ブレースを配置し、層間変形・偏心を抑える。
- **アトリウム**：2階床レベル（吹抜上部）に **スパン18m の大梁**が架かる。
  当初案は溶接組立 H 形断面（BH-1000 級）の単純大梁。ここが鉄骨重量・コストの集中点。
- **床**：デッキプレート＋現場打ちコンクリートスラブ（合成スラブではない普通スラブ）。小梁ピッチ 2.4m。
- **鋼材**：主要部材 SN490B。
- **耐火**：柱・梁とも吹付けロックウール耐火被覆。一部露出柱あり。

### 主要数量

| 項目 | 数量 |
| --- | --- |
| 鉄骨総重量 | **約 1,300 t**（≒ 108 kg/m²） |
| 鉄骨単価（製作・建方込 概算） | 約 35 万円 / t |

## 3. 工事費の内訳（概算）

| 区分 | 金額 | 備考 |
| --- | --- | --- |
| 総工事費 | 約 24.0 億円 | @20万円/m² |
| └ **躯体工事費** | **約 7.0 億円** | VE の対象 |
| 　├ 鉄骨工事 | 約 4.55 億円 | 1,300t × 35万円/t |
| 　├ 基礎・地下・スラブRC | 約 1.80 億円 | |
| 　└ 耐火被覆・その他躯体 | 約 0.65 億円 | |

> **VE目標 = 躯体工事費の1割 ≒ 0.70 億円**。
> 鉄骨数量に換算すると、**鉄骨1t ≒ 躯体費の0.05%**（35万円/7.0億円）。
> 例）鉄骨を100t 減らせば 3,500万円 ≒ 躯体費の5.0%に相当。

## 4. VE 要求（施主指示）

> 経営会議で総事業費の圧縮方針が出た。**構造躯体で1割（約0.7億円）削減できないか**検討してほしい。
> ただし次は守ること。
> 1. **正面アトリウムの無柱・吹抜は本ビルの「顔」。デザインは維持する。**
> 2. **工期は延ばせない**（テナント入居日確定）。
> 3. 安全性・法適合は当然厳守。

## 5. 制約・前提条件

- 構造計算ルート・適判：ルート3（保有水平耐力計算）を想定。法適合は絶対条件。
- 敷地・地盤：支持層 GL−22m で確定（追加調査の余地は本課題では無し）。
- 鉄骨ファブの製作能力・揚重機の能力には上限がある（大型部材は分割が必要になり得る）。
- 設計変更は基本設計の範囲内。確認申請前の段階とする。

## 6. この案件の「対立の構図」（ヒント）

```
        施主：1割減 必達／でもデザインも工期も守れ
                          │
   ┌──────────┼───────────┐
 意匠            設備             施工
 無柱死守      梁せい増やすな    工期・施工性
 （顔を守る）  （ダクト空間）   （複雑構造はリスク）
   └──────────┼───────────┘
                          │
        構造：安全は不可侵／削減案は複数出せる
```

各者の主張は一見両立しない。**前提（無柱は本当に死守か？ 梁せいは本当に増やせないか？）を会議で検証し、
隠れた妥協余地を引き出せるか**が合意の鍵になる。
