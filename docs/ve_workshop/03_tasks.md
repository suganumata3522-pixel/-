# 課題 ― 個人ワークと調整会議

## 課題① 個人ワーク：VE 案を出す（会議前・各自）

施主の「躯体1割減（≒0.7億円 ≒ 鉄骨換算 約130t）」に対し、構造設計者として **VE 案を5つ以上**出す。
各案について次を表にまとめること。

| 項目 | 記入内容 |
| --- | --- |
| 案の内容 | 何をどう変えるか（架構・部材・材種・床・基礎・耐火 等） |
| 削減効果 | 鉄骨数量（t・%）またはコスト（万円・躯体費%）で定量化 |
| 種別 | **数量VE** / **コストVE（単価・仕様）** / **施工VE（手間・工期）** のどれか |
| 安全性への影響 | たわみ・床振動・保有水平耐力・耐火 などへの波及 |
| 副作用・調整先 | 意匠／設備／施工のうち誰と何を調整する必要があるか |

> ヒント：削減の「軸」は数量だけではない。**鋼材単価・施工手間・工期**も躯体コストを動かす。
> 1案で1割に届く銀の弾丸はない。**組合せで目標に積み上げる**前提で考えること。

### 課題①の問い

1. 出した案を、**効果が大きい順**に並べよ。
2. そのうち「**意匠・設備・施工と調整が必要な案**」と「**構造単独で決められる案**」を仕分けよ。
3. **無柱アトリウムを維持したまま**1割に近づける組合せを1つ作れ。何%まで届くか。
4. 3で1割に届かない場合、**残りをどう埋めるか**（前提の再交渉／他工種VE）を1つ提案せよ。

## 課題② 役割の準備（会議前・各自）

`02_roles.md` で自分の役を読み込み、次を準備する。

- 自分の**公開の主張**（会議の冒頭で言うこと）
- 自分の**譲れない線**と**実は譲れる線**（非公開の利害）
- 相手から**引き出したい情報**は何か（例：構造役なら「設備は本当に梁せいを増やせないのか？」）

---

## 課題③ 調整会議（ロールプレイ・40〜60分）

施主の1割要求をテーブルに乗せ、合意案を作る。進行の目安：

### ステップ

1. **施主が要求を提示**（1割減・デザイン維持・工期厳守）。〔5分〕
2. **構造が VE メニューを提示**：個人ワークの案を、効果と副作用つきで説明。〔10分〕
3. **各役が反応**：意匠・設備・施工がそれぞれの制約を主張。ここで**対立が表面化**する。〔10分〕
4. **前提の検証と代替案づくり**：
   - 無柱は本当に死守か？（意匠・施主に確認）
   - 梁せいは本当に増やせないか？（設備に確認）
   - 工期を縮める施工VEはないか？（施工に確認）
   - 構造は安全余裕をどこまで攻められるか？
   〔15〜25分〕
5. **施主が裁定し、合意案をまとめる**。〔10分〕

### 会議で必ず詰める論点

- [ ] 無柱維持ルートと、柱を意匠化するルートの**両方を俎上に載せたか**
- [ ] 各 VE 案の削減%を**積み上げて目標との差を可視化**したか
- [ ] 梁せい・ダクトの対立に、**ルート変更などの第3案**を出せたか
- [ ] 施工VE（ユニット化・工期）を**構造案と連動**させたか
- [ ] 安全性を犠牲にしていないことを**定量で確認**したか
- [ ] 「1割に届かない分」をどう扱うか、**施主の優先順位を引き出して**合意したか

---

## 課題④ アウトプット（会議後・チーム）

合意内容を次の3点でまとめ、発表する。

1. **合意した VE 案の一覧**（案・削減%・調整した相手）と**到達した削減率**
2. **未解決事項・残ったリスク**（例：床振動の追加検討、ファブ製作能力の確認）
3. **施主への説明 1 枚**：
   「なぜこれだけ削れるのか／なぜこれ以上は削れないのか」を、
   **非専門家にわかる言葉で**、安全性の担保とセットで説明する。

> この「1枚」が、4つの力のうち**説明する力**の最終成果物。
> 数式ではなく「どこに余裕があり、どこは限界か」を語れるかが評価される。
