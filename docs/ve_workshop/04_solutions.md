# 解説（進行役用）

正解は一つではない。以下は**模範的な思考プロセスと到達点**であり、これと違う合意でも、
根拠が説明でき・安全で・合意が取れていれば妥当とする。

---

## 1. VE メニュー（模範解の素材）

当初鉄骨 1,300t、単価 35万円/t、躯体費 7.0億円。**鉄骨1t ≒ 躯体費0.05%**。

| 案 | 内容 | 種別 | 鉄骨削減 | 躯体費効果 | 主な副作用・調整先 |
| --- | --- | --- | --- | --- | --- |
| **A** 中間柱追加 | アトリウム18m→9m+9m に柱1本 | 数量 | ▲65t (5.0%) | ▲約3.2% | **無柱コンセプト崩壊**（意匠・施主） |
| **B** 張弦梁化 | アトリウム大梁を張弦梁(BSS)に | 数量 | ▲33t (2.5%) | ▲約1.6% | 下弦材・束の納まり、施工手間（施工） |
| **C** 合成梁 | 基準階梁をスタッド合成梁に | 数量 | ▲26t (2.0%) | ▲約1.3% | ほぼ無し（**推奨**）。スタッド施工 |
| **D** 高強度鋼 | 大スパン梁柱に SA440 部分採用 | 数量+単価 | ▲20t (1.5%) | ▲約0.5%※ | 単価UPで正味薄、溶接管理（施工） |
| **E** グリッド調整 | 基準階スパン割の見直し | 数量 | ▲13t (1.0%) | ▲約0.6% | 意匠グリッド・レイアウト（意匠） |
| **F** 小梁VE | 小梁ピッチ拡大＋合成デッキ | 数量 | ▲10t (0.8%) | ▲約0.5% | たわみ・床振動の再検討（構造） |
| **G** 耐火被覆VE | 露出柱を耐火塗料化 等 | コスト | ― | ▲約1.5% | 維持管理・納まり |
| **H** 基礎VE | 杭径・本数・長さの最適化 | コスト | ― | ▲約1.2% | 支持力検証、地盤リスク |
| **I** 施工VE | 鉄骨ユニット化・工場製作比率UP | 施工 | ― | ▲約1〜2% | 建方費・仮設費減、工期も短縮（施工発） |

※ D は鉄骨は減るが SA440 の単価が SN490 の約1.5倍。**正味のコスト効果は数量効果の1/3程度**になる点が「本質理解」の山場。

## 2. 模範的な到達点（2ルート）

### ルートⅠ：無柱を維持して攻める（王道）

| 採用案 | 躯体費効果 |
| --- | --- |
| B 張弦梁（アトリウム） | ▲1.6% |
| C 合成梁 | ▲1.3% |
| E グリッド調整 | ▲0.6% |
| F 小梁VE | ▲0.5% |
| G 耐火被覆VE | ▲1.5% |
| H 基礎VE | ▲1.2% |
| **小計** | **▲6.7%** |
| ＋ I 施工VE（ユニット化） | ▲1〜2% |
| **合計** | **約 8〜9%** |

- **無柱は B（張弦梁）で死守**。1割にはわずかに届かない（8〜9%）。
- ここで施主の**真の優先順位（コスト＞工期＞意匠細部、7〜8%で吸収可）を引き出せれば**、
  「無柱維持で約8%」を**合意**に持ち込める。これが調整力の到達点。
- D（高強度鋼）は正味効果が薄いので**無理に積まない**判断ができると加点。

### ルートⅡ：柱を意匠化して攻める（前提を疑えた場合のみ妥当）

会議で「無柱は本当に死守か？」を検証し、**施主・意匠が「柱1本を意匠的シンボルにするなら可」と認めた**場合：

| 採用案 | 躯体費効果 |
| --- | --- |
| A 中間柱（意匠化した1本） | ▲3.2% |
| C 合成梁 | ▲1.3% |
| E グリッド調整 | ▲0.6% |
| G 耐火被覆VE | ▲1.5% |
| H 基礎VE | ▲1.2% |
| F 小梁VE | ▲0.5% |
| **合計** | **約 8.3%** ＋ I で **10%到達** |

- A は単独で 3.2% と効果絶大。**目標1割に最短で届く**。
- ただし**構造の都合だけで顔を壊すのは✕**。「無柱を諦める」のではなく「**柱を意匠資源に変える**」合意を、意匠・施主と取れていることが条件。

> どちらのルートも「**1案では届かない／ポートフォリオで積む**」「**数量VEとコスト・施工VEを区別する**」が共通の学び。

## 3. トレードオフの整理

```
            コスト減
              ▲
   施工VE ─── │ ─── 数量VE（B/C/E/F）
   (工期↓)    │      安全余裕の範囲で
              │
 ─────────────┼───────────── 安全性（不可侵の床）
              │
   意匠(無柱) │ 設備(梁せい)
   ＝制約      │  ＝制約 … 第3案(柱意匠化/ダクト・ルートB)で緩和
```

- **安全性は床（下限）であって調整対象ではない**。削るのは「取りすぎた余裕」まで。
- 意匠・設備の制約は**固定ではなく、第3案で動かせる**（柱の意匠化、ダクトのルートB）。
- 工期は施工VEで**むしろ味方にできる**（ユニット化はコストと工期を同時に改善）。

## 4. 講評ポイント（4つの力ごと）

| 力 | 良い兆候 | ありがちな失敗 |
| --- | --- | --- |
| **応用** | 数量・単価・施工の3軸でVEを出した。Dの正味効果の薄さに気づいた | 「鉄を細くする」発想だけ。単価・工期の軸が抜ける |
| **判断** | 1案で届かないと見て組合せで積んだ。安全余裕の範囲を意識 | 効果の大きいAに飛びつき、コンセプト崩壊に無自覚 |
| **調整** | 「無柱は死守か」「梁せい増は不可か」と**前提を検証**し、隠れ制約を引き出した | 各自が自分の主張を繰り返すだけ。第3案が出ない |
| **説明** | 「どこに余裕があり、どこが限界か」を施主の言葉で説明 | 「計算上OKです」で終わり、根拠を翻訳できない |

## 5. 進行役の介入例（会議が詰まったら）

- 対立が膠着したら：「**その制約は何があれば動きますか？**」と各者に問わせる。
- Aに安易に飛びついたら：「それは施主が一番大事にしている『顔』ですが、誰が施主に説明しますか？」
- 数字が出てこないなら：「いま**何%まで積めて、目標までいくら残っていますか?**」と可視化を促す。
- 全員が遠慮して進まないなら、施主役に「**1割未満なら他で吸収できる**」のカードを**最後まで切らせない**（先に出すと訓練価値が下がる）。

## 6. 評価ルーブリック（オブザーバ用）

各軸 3段階。チーム発表とプロセス観察で評価する。

| 軸 | Lv1 | Lv2 | Lv3 |
| --- | --- | --- | --- |
| **応用** | VE案が数量中心で2〜3個 | 5案以上、コスト/施工も含む | 各案を定量化し、Dの罠など質的差を説明 |
| **判断** | 単案頼み or 安全度外視 | 組合せで目標に接近 | 安全余裕の範囲を意識し優先順位で取捨 |
| **調整** | 主張の応酬で終わる | 一部の隠れ制約を引き出した | 前提を検証し第3案でWin-Winを作った |
| **説明** | 専門用語のまま | 削減根拠は言える | 根拠＋安全担保を非専門家に1枚で説明 |

> 満点の合意より、**前提を疑い・隠れ制約を引き出し・トレードオフを言語化できたか**を高く評価する。
> 中堅に最も欠けがちなのは「与えられた前提（無柱は死守）をそのまま受け入れてしまう」こと。
> それを**会議の場で検証しにいけたか**が、この勉強会の最大の見どころ。
