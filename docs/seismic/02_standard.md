# 02. 標準問題

実務に直結する 4〜6 階建ての全層算定。
全問とも C0 = 0.2（一次設計）で計算する。

---

## 問題 2-1 （RC造 4階建、第2種地盤、東京）

### 条件

- 構造：純RC造
- 軒高：h = 16.0 m（各階 4.0 m × 4層）
- 地域：東京（Z = 1.0）
- 地盤種別：第2種（Tc = 0.6 s）
- 各階重量（地震用重量 Wj）：

| 階 | Wj (kN) |
| --- | --- |
| 4階 | 4,500（屋根含む） |
| 3階 | 5,000 |
| 2階 | 5,000 |
| 1階 | 5,500 |
| 合計 W | 20,000 |

### 設問

(1) 設計用一次固有周期 T を求めよ。
(2) Rt を求めよ。
(3) 各層の αi、Ai、Ci、Qi、Pi を表にまとめよ（4層〜1層）。

---

## 問題 2-2 （S造 5階建、第1種地盤、大阪）

### 条件

- 構造：純S造
- 軒高：h = 20.0 m（各階 4.0 m × 5層）
- 地域：大阪（Z = 1.0）
- 地盤種別：第1種（Tc = 0.4 s）
- 各階重量：

| 階 | Wj (kN) |
| --- | --- |
| 5階 | 2,800 |
| 4階 | 3,200 |
| 3階 | 3,200 |
| 2階 | 3,200 |
| 1階 | 3,600 |
| 合計 W | 16,000 |

### 設問

(1) T、Rt を求めよ。
(2) 各層の αi、Ai、Ci、Qi、Pi を求めよ。
(3) 同建物を東京 → 静岡（Z = 1.0）→ 福岡（Z = 0.8）に建てた場合、1層の Q1 はそれぞれいくらか。

---

## 問題 2-3 （RC造 6階建、第3種地盤、地震多発地域）

### 条件

- 構造：純RC造
- 軒高：h = 21.0 m（各階 3.5 m × 6層）
- 地域：Z = 1.0
- 地盤種別：第3種（Tc = 0.8 s）
- 各階重量：

| 階 | Wj (kN) |
| --- | --- |
| 6階 | 6,000（屋根含む） |
| 5階 | 7,000 |
| 4階 | 7,000 |
| 3階 | 7,000 |
| 2階 | 7,000 |
| 1階 | 8,000 |
| 合計 W | 42,000 |

### 設問

(1) T、Rt を求めよ。
(2) 各層の αi、Ai、Ci、Qi、Pi を求めよ。
(3) 同じ建物を第1種地盤（Tc = 0.4 s）に建てた場合、Rt と 1層の Q1 を比較せよ（軟弱地盤の方が地震力が大きい理由を考察）。

## 問題 2-4 （地域係数の影響）

問題 2-1 の RC 4階建を、以下の3地域に建設する場合の 1層 Q1 を比較せよ。

| 地域 | Z |
| --- | --- |
| 東京 | 1.0 |
| 札幌 | 0.9 |
| 沖縄 | 0.7 |

Q1 の比率（東京を 1.0 とした場合）も併記すること。
