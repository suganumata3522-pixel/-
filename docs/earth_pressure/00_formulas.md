# 00. 土圧 基本式・記号リファレンス

ゼネコン構造設計部・新人向け。
擁壁・地下外壁・山留め壁に作用する土圧の算定に用いる。
建築基礎構造設計指針、道路土工 擁壁工指針、土質力学の基礎に基づく。

土質定数（γ、φ、c、N 値との関係）は**地盤調査報告書（ボーリング柱状図・土質試験）を手元に置いて引く**こと。
本書では式と考え方のみ示す。

## 1. 体系図（3種類の土圧）

```
        壁の動き                 土圧係数        大小関係
┌────────────────────┬──────────────┬──────────────┐
│ 壁が動かない（静止）│ 静止土圧 K0  │              │
│ 壁が土から離れる    │ 主動土圧 Ka  │ Ka < K0 < Kp │
│ 壁が土を押し込む    │ 受動土圧 Kp  │              │
└────────────────────┴──────────────┴──────────────┘
```

| 種類 | 記号 | 発生条件 | 主な設計対象 |
| --- | --- | --- | --- |
| 静止土圧 | K0 | 壁が変位しない（剛な拘束） | 地下外壁、ボックスカルバート、剛強な壁 |
| 主動土圧 | Ka | 壁が背面土から離れる向きに微小変位 | 擁壁の転倒・滑動、山留め壁 |
| 受動土圧 | Kp | 壁が背面土を押し込む向きに変位 | 根入れ前面抵抗、アンカー抵抗 |

## 2. 土圧係数の式

### 静止土圧係数 K0（ヤーキの式）
```
K0 = 1 − sin φ        （正規圧密の砂・粘土）
```
過圧密土では K0 はこれより大きくなる（OCR の影響）。

### 主動・受動土圧係数（ランキン式、壁面摩擦・傾斜なし）
```
Ka = tan²(45° − φ/2) = (1 − sin φ)/(1 + sin φ)
Kp = tan²(45° + φ/2) = (1 + sin φ)/(1 − sin φ)
```
ランキン式では `Kp = 1 / Ka` の関係が成り立つ。

### 主動土圧係数（クーロン式、壁面摩擦・傾斜あり）
```
            cos²(φ − θ)
Ka = ───────────────────────────────────────────────
      cos²θ · cos(δ + θ) · [ 1 + √R ]²

           sin(φ + δ) · sin(φ − β)
      R = ─────────────────────────
           cos(δ + θ) · cos(θ − β)
```

| 記号 | 意味 |
| --- | --- |
| φ | 土の内部摩擦角（°） |
| δ | 壁面摩擦角（°）。一般に δ = (1/2〜2/3)·φ |
| θ | 壁背面の鉛直からの傾斜角（°）。鉛直壁で θ=0 |
| β | 地表面（背面地盤）の水平からの傾斜角（°）。水平で β=0 |

θ=0、δ=0、β=0 のときクーロン式はランキン式に一致する。

## 3. 代表的な φ と土圧係数（目安）

| φ | K0=1−sinφ | Ka=tan²(45−φ/2) | Kp=tan²(45+φ/2) |
| --- | --- | --- | --- |
| 25° | 0.58 | 0.41 | 2.46 |
| 30° | 0.50 | 0.33 | 3.00 |
| 35° | 0.43 | 0.27 | 3.69 |
| 40° | 0.36 | 0.22 | 4.60 |

φ は砂質土で N 値から推定（例：φ ≒ √(15N) + 15、または √(20N) + 15）。粘性土は別途 c で評価。

## 4. 土圧（圧力・合力・作用点）

深さ z における主動土圧（単位幅あたり）：
```
pa(z) = Ka · γ · z + Ka · q        [kN/m²]
        └ 自重分（三角形分布）┘ └ 上載分（矩形分布）┘
```

| 成分 | 圧力分布 | 合力（壁高 H） | 合力作用点（壁下端から）|
| --- | --- | --- | --- |
| 自重土圧 | 三角形（0 → KaγH）| `½ · Ka · γ · H²` | `H/3` |
| 上載土圧 | 矩形（一様 Kaq）| `Ka · q · H` | `H/2` |

合力の作用点は両成分の**モーメント加重平均**で求める。

粘着力 c がある土（c-φ 土）：
```
pa(z) = Ka · γ · z − 2c · √Ka
```
上部で pa が負（引張）になる深さ `zc = 2c/(γ√Ka)` までは土が壁から離れる
（引張は無視＝クラックを想定し、合力に算入しないのが安全側）。

## 5. 上載荷重 q

- 背面地表に作用する等分布荷重。土圧を一様に `Ka·q`（矩形分布）だけ増やす
- 土被り換算：`q = γ · h` 相当（厚さ h の土層に置き換えられる）
- 標準値の目安（指針・条例で確認）：

| 用途 | q の目安 (kN/m²) |
| --- | --- |
| 一般地表面・宅地 | 10 |
| 駐車場・通路 | 10〜12 |
| 重車両（消防車進入路等）| 12〜20 |
| 仮設工事中（建機・資材）| 別途、施工計画で設定（短期）|

- 載荷位置が壁から離れた線・点荷重は、ブーシネスク式等で深さ別の応力増分を評価（一様にならない）

## 6. 地下水位がある場合

水位を境に**単位体積重量を切り替え**、**水圧を別途加算**する。

| 領域 | 土圧に使う γ | 備考 |
| --- | --- | --- |
| 地下水位より上 | γt（湿潤単位体積重量）| |
| 地下水位より下 | γ' = γsat − γw（水中単位体積重量）| γw ≒ 9.8 kN/m³ |

```
全側圧 = 有効土圧（Ka·σ'v + Ka·q） + 水圧（pw = γw · z'）
```
- `σ'v`：有効鉛直応力（水位以下は γ' を累積）
- `z'`：地下水位からの深さ
- 水圧は静水圧として三角形分布で別に集計（Ka は掛けない）

## 7. 計算手順（実務フロー：擁壁の主動土圧）

1. ボーリングより各層の **γ・φ・c・地下水位** を整理
2. 壁の変位条件から **K0 / Ka / Kp** を選択（擁壁は通常 Ka）
3. φ と壁形状から **土圧係数**（ランキン or クーロン）を算出
4. **上載荷重 q** を用途から設定
5. 深さ z ごとに **pa(z)** を作図（自重三角形 + 上載矩形、水位で勾配変化）
6. **合力 Pa** と **作用点** を算出（地下水位があれば水圧 Pw も別途）
7. 合力の水平・鉛直分力 →（擁壁なら）**転倒・滑動・支持**の安定計算
8. 部材（地下外壁・擁壁竪壁）の **曲げ・せん断**の設計

## 8. よくあるミス

- 擁壁なのに **K0 で過大設計**、地下剛壁なのに **Ka で過小（危険側）** ── 変位条件で係数を選ぶ
- クーロン式で `(45°−φ/2)` と `(45°+φ/2)` を取り違える（Ka と Kp が逆転）
- 角度を **degree のまま三角関数に入れる**（電卓のモード／単位に注意）
- 上載土圧の作用点を H/3 にしてしまう（**矩形分布なので H/2**）
- 地下水位以下で γt のまま計算（**γ' に切替**＋**水圧を別加算**を忘れる）
- 水圧に Ka を掛けてしまう（水は静水圧、Ka は掛けない）
- 粘着力の引張側（上部の負圧）を合力に算入（**引張無視が安全側**）
