# 03. 応用問題

実務でよく遭遇する複雑なケース。

---

## 問題 3-1 （混構造の固有周期）

10階建ての建物で、1〜3階がRC造（h = 12 m）、4〜10階がS造（h = 21 m）となっている。
建物全体の高さ h = 33 m。

(1) この建物の鉄骨造高さ比 α を求めよ。
(2) 設計用一次固有周期 T を求めよ。
(3) この T が、純RC造 33 m、純S造 33 m と比較してどちらに近いか述べよ。

---

## 問題 3-2 （重量の偏りがある建物）

3階建てだが、3階に重量物（機械室）があり、各階重量が以下のようになっている。

| 階 | Wj (kN) |
| --- | --- |
| 3階 | 8,000（機械室） |
| 2階 | 3,000 |
| 1階 | 3,000 |
| 合計 W | 14,000 |

条件：純RC造、各階高 4.0 m、h = 12 m、Z = 1.0、第2種地盤。

(1) T、Rt を求めよ。
(2) 各層の Ai を求めよ。
(3) 通常の重量配分（各階 4,667 kN ずつ）の場合と比較して、3層 Ai はどう変化するか定性的に説明せよ。

---

## 問題 3-3 （Rt の場合分けの境界）

T と Tc の関係で Rt の式が切り替わる。次の場合の Rt を求め、計算経路を示せ。

(1) T = 0.40 s、第2種地盤（Tc = 0.6 s）→ どの区間か？
(2) T = 0.60 s、第2種地盤（Tc = 0.6 s）→ 境界値、Rt = ?
(3) T = 1.20 s、第2種地盤（Tc = 0.6 s）→ 境界値（2Tc）、Rt = ?
(4) T = 1.21 s、第2種地盤（Tc = 0.6 s）→ どの区間か？
   ※ 境界の前後で Rt が連続することを確認せよ。

## 問題 3-4 （許容応力度設計と保有水平耐力の比較）

問題 2-1 の RC 4階建てに対して、必要保有水平耐力を求める際の地震層せん断力を計算せよ。

条件：
- 必要保有水平耐力時の C0 = 1.0
- 構造特性係数 Ds = 0.40（仮定）
- 形状係数 Fes = 1.0（均整な建物として仮定）

(1) 必要保有水平耐力時の各層 Qud を求めよ。
(2) 必要保有水平耐力 Qun = Ds · Fes · Qud を各層で求めよ。
(3) 一次設計時 Q（問題 2-1）と Qud の比率を確認せよ。

## 問題 3-5 （実務総合問題：8階建てオフィスビル）

### 条件

- 構造：純S造（純鉄骨造）
- 規模：8階建て
- 階高：1階 4.5 m、2〜8階 3.8 m → h = 4.5 + 3.8 × 7 = 31.1 m
- 地域：横浜（Z = 1.0）
- 地盤種別：第2種（Tc = 0.6 s）
- 各階重量（地震用、単位 kN）：

| 階 | Wj |
| --- | --- |
| R階(=8階屋根扱い) | ― |
| 8階 | 4,200（屋根重量含む） |
| 7階 | 4,800 |
| 6階 | 4,800 |
| 5階 | 4,800 |
| 4階 | 4,800 |
| 3階 | 4,800 |
| 2階 | 4,800 |
| 1階 | 5,400 |
| 合計 W | 38,400 |

### 設問

(1) T、Rt を求めよ。
(2) 各層の αi、Ai、Ci、Qi、Pi を表にまとめよ。
(3) 地震時にこの建物が基礎に伝える総水平力（ベースシア） Q1 と、ベースシア係数 Q1/W を確認せよ。
(4) C0 = 0.2 · Z · Rt（= 平均的な一次設計時 Cb） と Q1/W が等しいことを確かめよ（Ai 分布の重み付き平均が 1 になることを意味する）。

## 問題 3-6 （考察問題）

(1) なぜ最上層の Ai が大きく、1層の Ai = 1 となるのか、力学的に説明せよ。
   ヒント：地震時の建物変形モード（1次モード）と慣性力分布。

(2) 設計用一次固有周期 T が長くなるほど Ai が「上層集中型」になる。
   この理由を、Ai 式の `(2T)/(1+3T)` の項に着目して説明せよ。

(3) 地盤が軟弱（第3種、Tc 大）になるほど、長周期建物の Rt は小さくなる。
   一方で短周期建物の Rt はどうなるか、設計上どのような注意が必要か述べよ。
